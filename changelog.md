
# V1.1
### Core
1. 接入平台存储 SDK，支持数据的跨环境持久化（AI Studio 对应 localStorage，平台部署环境对应云端存储）。
2. 重构 LLM 调用逻辑，支持通过平台代理（/api/llm/proxy）进行大模型调用，提升安全性和集成度。
3. 适配 Gemini 3 API 规范，完善角色定义（Role: user/model）。

### Feature
1. 增加环境感知功能，侧边栏显示当前用户的平台个人资料或本地状态。
2. 完善 AI 错误处理机制，针对 401 认证失败提供人性化提示。

### UI/UX
1. 优化应用初始化状态，增加 Loading 动效。
2. 调整侧边栏用户模块视觉效果，根据平台用户信息动态展示头像占位。

# V1.0 
### Core
1. 基础 React 架构搭建，集成 Tailwind CSS。
2. 实现基础笔记增删改查功能。
3. 集成 Gemini Flash 模型进行摘要、扩写、纠错及标签推荐。

### Feature
1. 瀑布流（Masonry）布局展示笔记。
2. 颜色标记、星标收藏功能。
3. 实时标签过滤。

### UI/UX
1. 采用 Inter 字体，Indigo 主题色。
2. 沉浸式全屏笔记编辑器。
